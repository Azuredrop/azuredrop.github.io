Markdown使用技巧(进阶)

  • AzureDrop
  • 2018/07/12
  • 阅读数 次
  • 编程, Markdown
Category: Categories: 编程

1.文字链接

  使用[显示文本](链接地址) 格式。
写法： 

[百度](www.baidu.com)

效果：
百度

2.图片链接

  使用![显示文本](链接地址) 格式。
写法： 

![百度](https://www.baidu.com/img/bd_logo1.png)

效果：

3.列表

a.无序列表

  可以在前面使用“*”、“+”或者“-”。
写法： 

* 无序列表项1  
+ 无序列表项2  
- 无序列表项3

效果：

  • 无序列表项1
  • 无序列表项2
  • 无序列表项3

b.有序列表

写法： 

1.有序列表项1  
2.有序列表项2  
3.有序列表项3

效果：

1.有序列表项1
2.有序列表项2
3.有序列表项3

4.表格

写法： 

| 表头1|表头2|表头3|表头4|  
|-| :- | :-: | -: |  
|默认左对齐AAA|左对齐AAA|居中对其AAA|右对齐AAA|  
|默认左对齐B|左对齐B|居中对其B|右对齐B|  
|默认左对齐CC|左对齐CC|居中对其CC|右对齐CC|

效果：

表头1 表头2 表头3 表头4
默认左对齐AAA 左对齐AAA 居中对其AAA 右对齐AAA
默认左对齐B 左对齐B 居中对其B 右对齐B
默认左对齐CC 左对齐CC 居中对其CC 右对齐CC

5.代码块

a.无行号

  在前面输入4个空格或者使用制表符
效果： 

代码块-4个空格  
代码块-1个制表符

b.有行号

  在首位用三个“`”括起来
效果：

6.待办事项（To-do List）

写法： 

- [x] 已完成项目1
    - [x] 已完成项目2
    - [x] 已完成项目3
- [ ] 待办项目1
- [ ] 待办项目2

效果：

  • 已完成项目1
    • 已完成项目2
    • 已完成项目3
  • 待办项目1
  • 待办项目2
